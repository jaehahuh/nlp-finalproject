## 🖥️ Project Introduction
|**개요**|**Description**|
|:--:|--|
|**주제** | 우울증 예측 프로젝트 |
|**목표**| 기존의 PHQ-9과 같은 설문 형식의 자가진단과 달리, 사람의 목소리와 발화 내용으로부터 패턴을 추출해 우울증 여부를 예측하는 모델 학습 및 서비스 배포 |
|**평가 지표**| Recall |
|**데이터**| 8가지 질문에 대한 참여자의 발화 내용과 우울증 여부 및 나이, 성별 등 기본 인적사항이 포함된 총 18000개의 데이터 |
